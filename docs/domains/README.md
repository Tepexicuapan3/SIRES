# Dominios - Documentacion Canonica

Este indice agrupa la documentacion por dominio funcional para escalar mantenimiento, ownership y trazabilidad.

## Dominios disponibles

- [`auth-access/README.md`](./auth-access/README.md) - Hub canonico del Dominio 1 (autenticacion, identidad, autorizacion, recuperacion y trazabilidad).

## Convenciones

- Cada dominio tiene un `README.md` como hub unico.
- Los documentos tematicos viven dentro de su carpeta de dominio.
- Las rutas legacy en `docs/guides/` pueden mantenerse temporalmente como stubs de redireccion (deprecados) para no romper discoverability.
